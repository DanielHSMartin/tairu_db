TairuDB Plugin para QGIS

Plugin para geração de arquivos .tairudb compatíveis com o aplicativo móvel Tairu Maps (iOS, Android e Web). Permite exportar mapas offline — incluindo tiles raster e camadas vetoriais — diretamente de projetos QGIS.

O que é o Tairu Maps?

O Tairu Maps é um aplicativo móvel de mapas colaborativos voltado para uso em campo, com suporte a mapas offline, compartilhamento de localização em tempo real, registros georreferenciados e integração com dados geográficos produzidos no QGIS. O arquivo .tairudb é o formato nativo de importação de mapas offline no Tairu Maps.

O arquivo .tairudb

Um arquivo .tairudb é um banco de dados SQLite3 com as seguintes tabelas:

TabelaConteúdo
metadataConfigurações gerais (formato, zoom, centro, versão)
regionsRegiões geográficas com bounds e níveis de zoom
tiles_region_NTiles raster em formato TMS por região (PNG, JPG ou WebP)
vector_layersCamadas vetoriais exportadas (ponto, linha, polígono)
featuresFeições com geometria, estilo e atributos em JSON

Como usar

  1. Abra o QGIS com um projeto contendo as camadas raster desejadas e deixe-as visíveis.
  2. Crie ou importe uma camada vetorial com o(s) polígono(s) delimitando a(s) área(s) de interesse.
  3. Acesse Processamento > Caixa de Ferramentas e procure por TairuDB.
  4. Preencha os parâmetros e clique em Executar.
  5. Transfira o arquivo .tairudb gerado para o seu dispositivo e importe-o no Tairu Maps.

Parâmetros

ParâmetroDescrição
Área de interesse (polígono)Camada vetorial de polígonos. Cada feição gera uma região independente no arquivo de saída.
Resolução do mapaDefine o nível de zoom máximo: de Altíssima (0,5 m/px, zoom 18) a Muito Baixa (32 m/px, zoom 12).
Formato da imagemPNG (sem perda), JPG ou WebP. JPG e WebP produzem arquivos menores com perda de qualidade configurável.
QualidadeQualidade de compressão para JPG/WebP (1–100). Padrão: 90.
Camadas vetoriais para exportar(Opcional) Camadas vetoriais do projeto a serem incluídas no arquivo, com cor e atributos.
Arquivo de saídaCaminho para o arquivo .tairudb a ser gerado.

Conversor GeoPDF

O arquivo geopdf_converter.py permite converter arquivos GeoPDF para o formato .tairudb, extraindo o fundo raster como tiles e as geometrias vetoriais como camadas. Requer GDAL, pyproj e as bibliotecas Python do QGIS. 

python geopdf_converter.py entrada.pdf saida.tairudb [opções]

Recomendações de resolução

ResoluçãoZoomUso indicado
Altíssima (0,5 m/px)18Plantas, levantamentos de detalhe
Alta (1 m/px)17Áreas urbanas, trilhas
Médio Alta (2 m/px)16Bairros, parques
Média (4 m/px)15Cidades pequenas
Médio Baixa (8 m/px)14Municípios
Baixa (16 m/px)13Regiões
Muito Baixa (32 m/px)12Estados, visão geral
Atenção: Resoluções altas em áreas grandes podem gerar milhares de tiles e demorar vários minutos. Avalie a relação entre tamanho da área e resolução antes de executar.
