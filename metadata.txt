about=TairuDB exports QGIS map data to .tairudb (SQLite), used by the Tairu Maps mobile app (iOS/Android). Stores raster tiles (PNG/JPG/WebP), vector layers with attributes, and navigation metadata. Run via Processing > Toolbox: set an area polygon, resolution (0.5–32 m/px), image format and optional vector layers, then import the file into Tairu Maps. Includes a GeoPDF converter.
  ---
  Exporta dados do QGIS para o formato .tairudb (SQLite), usado pelo aplicativo Tairu Maps (iOS/Android). Armazena tiles raster (PNG/JPG/WebP), camadas vetoriais com atributos e metadados de navegação. Execute via Processamento > Caixa de Ferramentas: defina um polígono de área, resolução (0,5–32 m/px), formato de imagem e camadas vetoriais opcionais, e importe o arquivo no Tairu Maps. Inclui conversor GeoPDF.
