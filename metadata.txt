about=TairuDB converts geographic data from QGIS projects into the .tairudb format used by the Tairu Maps mobile app (iOS and Android).
  A .tairudb file is a SQLite database that stores raster tiles in TMS format (PNG, JPG or WebP) organised by region, vector layers (points, lines and polygons) with attributes and styles, and navigation metadata.
  How to use: (1) Open a QGIS project with the desired raster layers visible; (2) Create or import a polygon defining the area of interest; (3) Run the TairuDB algorithm via Processing > Toolbox; (4) Set the area polygon, resolution (0.5 m/px to 32 m/px), image format and optional vector layers; (5) Import the generated .tairudb file into the Tairu Maps app.
  Also includes a GeoPDF converter (geopdf_converter.py) to convert GeoPDF files directly to .tairudb format.
